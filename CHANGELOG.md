# Changelog

All notable changes to RealEstate Booking Suite will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added
- Initial release
- Property custom post type with extensive metadata
- Frontend property submission form
- Advanced AJAX search with multiple filters
- Property archive templates (grid, list, map view)
- Single property detail page template
- Interactive maps integration (Leaflet.js + OpenStreetMap)
- Booking management system
- Favorites/wishlist functionality
- Search alerts with email notifications
- Contact forms for property inquiries
- Email notification system
- Frontend user dashboard
- Admin dashboard with statistics
- Property management metabox
- Elementor widgets:
  - Property Grid Widget
  - Property Carousel/Slider Widget
  - Search Widget
  - Listings Widget
  - Request Form Widget
  - Authentication Widget
- User roles and capabilities system
- Badge system for properties
- Responsive design (mobile-first)
- Block theme compatibility
- Translation support (.pot file included)
- Security features (nonces, sanitization, escaping)
- AJAX-powered interactions
- Admin settings panel
- Contact messages management
- Email template management
- Demo content importer

### Security
- All user inputs sanitized
- All outputs escaped
- Nonce verification on all AJAX requests
- Capability checks for admin functions
- SQL injection protection with prepared statements
- XSS protection

### Performance
- Optimized database queries
- Efficient AJAX handlers
- Caching where appropriate

### Compatibility
- WordPress 5.8+
- PHP 7.4+
- MySQL 5.6+
- Elementor (optional)
- Block themes
- Classic themes
