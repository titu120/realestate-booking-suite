RealEstate Booking Suite - Documentation

Thank you for purchasing RealEstate Booking Suite. This documentation will guide you through the installation and configuration process.

1. Installation

  1. Log in to your WordPress admin dashboard.
  2. Go to Plugins > Add New.
  3. Click Upload Plugin.
  4. Choose the realestate-booking-suite.zip file and click Install Now.
  5. Activate the plugin.

2. Getting Started

Once activated, you will see a new menu item called RealEstate Booking Suite in your dashboard.

Tip: To quickly set up your site, go to RealEstate Booking Suite > Demo Import and click the "Import Demo Content" button.

3. Key Features

  • Properties: Add and manage property listings with detailed specifications.
  • Bookings: Manage booking requests and availability.
  • Agents: Assign agents to properties.
  • Settings: Configure currency, measurement units, and map settings.

4. Shortcodes

Use the following shortcodes to display content on your pages:

  • [resbs_properties] - Display a grid of properties.
  • [resbs_search_form] - Display the property search form.
  • [resbs_agents] - Display a list of agents.
